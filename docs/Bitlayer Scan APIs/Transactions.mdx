## Check Transaction Receipt Status
The status field returns 0 for failed transactionsï¼Œ1 for successful transactions and 2 for pending transactions.

```json
https://api.btrscan.com/scan/api
?module=transaction
&action=gettxreceiptstatus
&txhash=0x211beec218f9a19ba3acb440ecacbcd7d5bb1dfad7f591e24ffc999efb7f9d69
```
